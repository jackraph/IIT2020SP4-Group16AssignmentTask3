Tools and Technologies

The team is using a wide range of technologies for this project. Danielle is using paper to build paper prototypes. Timothy is prototyping in program called Pico-8, a virtual game console created to make and share tiny games. Jack is prototyping the game using Unity. Michael is building the level creator in Java and Eclipse.

Michael and Jack have experience working with unity, java and C#. Michael has a Diploma of Interactive Gaming and has worked in the Game development sector. Jack has used Unity for personal projects for 3 years. Both are doing a programming unit at RMIT focused on Java.

Timothy has used Java and PHP, but Lua and Pico-8 are new.

For Image editing the team will use Paint.Net and Krita (any versions). For video editing the team will use iMovie and QuickTime (any versions). (While we are using this software, any other similar software will do.)

When we make our game available online at indieDB or itch.io, we are required to make an account with each platform.

To inform our learning about game design, we used two resources: Nicole Lazzaro’s, 4Keys2Fun and Jeremy Schell’s The Elemental Tetrad.
